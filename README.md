> Ruby and Rails training course built as preparation for a Junior Back-end Developer position at Embarca.

Study repository documenting my Ruby and Ruby on Rails learning journey, built as part of my preparation for a **Junior Back-end Developer** position at **Embarca**.

## What's covered

| Module | Topics |
|--------|--------|
| 01 - Ruby Fundamentals | Variables, types, conditionals, loops, methods, blocks, arrays, hashes |
| 02 - Files & Gems | File I/O, CLI arguments, directories, gems (CSV, Faker, URI/HTTP) |
| 03 - Object-Oriented Programming | Classes, initialize, getters/setters, attr, inheritance, polymorphism |
| 04 - Databases | PostgreSQL with ActiveRecord |
| 05 - Advanced Ruby | Date/Time, Regex, Minitest (unit and class testing) |
| 06 - Git & GitHub | Version control, remote repositories, collaboration basics |
| 07 - Rails Fundamentals | MVC structure, models, controllers, routing, Active Record in depth |
| 08 - Authentication & Authorization | Devise, Pundit, admin role, model associations, foreign keys, migrations vs schema |
| 09 - RSpec | Test structure with describe/context/it, expectations with expect/eq, unit testing Ruby classes |
| 10 - Rails API | API-only Rails app (`--api` flag), JSON responses, CORS, routing for REST endpoints |

## Stack

- **Language:** Ruby
- **Framework:** Ruby on Rails
- **Databases:** PostgreSQL
- **Testing:** Minitest, RSpec
- **API testing:** Postman
- **Version control:** Git + GitHub
